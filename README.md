# 🔍 LostLink - Campus Lost & Found Platform
LostLink is a lightweight full-stack web application designed to help university campuses efficiently track and match lost items with their finders. Built with Python, Flask, and SQLite3, it provides an intuitive interface for students and staff to report lost and found items and view potential matches.

---

## 🚀 Features

* 📌 Report lost items
* 📌 Report found items
* 🔄 Real-time lost and found matching board
* 🗄️ SQLite3 database for persistent storage
* 📱 Responsive and user-friendly interface

---

## 🛠️ Tech Stack

* **Backend:** Python, Flask
* **Database:** SQLite3
* **Frontend:** HTML5, CSS3

---

## 💻 Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/malavikasanthikrishna06-coder/LostLink.git
cd LostLink
```

### 2. Install Dependencies

```bash
pip install flask
```

### 3. Run the Application

```bash
python app.py
```

### 4. Access the Application

After running the command, Flask will display output similar to:

```text
Running on http://127.0.0.1:5000
```

Open the following URL in your browser:

```text
http://127.0.0.1:5000
```

### 5. Stop the Application

Press `CTRL + C` in the terminal to stop the Flask server.
